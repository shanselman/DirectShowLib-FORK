VmrSnapper is an example of how to get a Bitmap from a video using VMR9.

This sample uses the following DirectShow Interfaces :

	IFilterGraph2
	IMediaControl
	IBaseFilter
	IVMRWindowlessControl9
	IVMRFilterConfig9

This sample is based on the BitmapMixer sample.
